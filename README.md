# cwy007.github.io

## 项目启动

```bash

cd /Users/chanweiyan/github/cwy007.github.io

# blogs='bundle exec jekyll serve --watch'
bundle exec jekyll serve --watch

# http://localhost:4000/


```

## 部署到 github page

- 在 _posts 目录下添加新的文章
- git commit
- git push 到 github
- 等一会刷新 `cwy007.github.io` 页面就会看到新的文章
