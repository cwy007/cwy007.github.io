## React 是什么？

* React是一个用来构建用户界面的库，它最初是由Facebook的工程师创建的。
* React提供了一套基于组件的简单、灵活的API。
* 组件是React的基本单元，在React应用中被广泛使用。
* React在程序和浏览器DOM之间实现了一个虚拟DOM层。
* 虚拟DOM使用快速比对算法对DOM进行高效更新。
* 虚拟DOM具有优秀的性能，但最大的好处是它提供的思维模型。
